üç∞ 3. Fractions

Understanding Fractions

3/4 = numerator (parts we have) / denominator (total parts)

Equivalent fractions = same value, different numbers

1/2 = 2/4 = 3/6 = 4/8
Write two equivalent fractions for 1/3

Adding & Subtracting Fractions

Same denominator = easy! Different denominator = find common denominator first.

1/4 + 1/4 = 2/4 = 1/2
1/2 + 1/4 = 2/4 + 1/4 = 3/4
1/3 + 1/3 = __________

Comparing Fractions

If denominators are same: larger numerator = larger fraction
If numerators are same: smaller denominator = larger fraction
3/5 vs 4/5: 4/5 is larger
Compare: 2/3 __________ 3/5

üìù 5. Word Problems

  2. Find the question (what do I need to find?)
  3. Find the numbers and information
